## 1. 程序简介
本项目是基于《人工智能：现代方法（第四版）》课程设计的AI程序，实现了A*搜索算法在经典八数码问题上的应用。系统提供了完整的算法实现、可视化界面和性能分析工具。

## 2. 文件结构
```
eight-puzzle-solver/
├── main.py              # 主程序入口
├── puzzle_state.py      # 状态表示类
├── a_star.py           # A*算法实现
├── puzzle_gui.py       # 图形用户界面
├── utils.py            # 工具函数
├── test_cases.py       # 测试案例和单元测试
├── collect_data.py     # 性能数据收集脚本
└── README.md           # 项目说明文档
```

## 3. 快速开始
### 3.1 环境要求
- Python版本: 3.8 或更高版本
- 操作系统: Windows / macOS / Linux
- 依赖库: tkinter (Python标准库，无需额外安装)
### 3.2 安装步骤
- 1.克隆或下载项目到本地
- 2.确保已安装python --version
- 3.所有依赖均为Python标准库，无需额外安装
### 3.3 运行程序
运行```python main.py```，选择“图形界面”模式。

## 4. 使用方法
### 4.1 启动程序
运行 python main.py，选择"图形界面"模式。
### 4.2 设置初始状态
- 预设案例：点击"简单"、"中等"、"困难"按钮加载测试案例
- 随机生成：点击"随机生成"创建随机可解状态
- 手动输入：在输入框中输入数字0-8（每个数字必须且只能出现一次）
### 4.2 选择算法参数
- 启发式函数：
- manhattan：曼哈顿距离（推荐，性能更优）
- misplaced：错位数（简单但效率较低）
### 4.3 开始求解
点击“开始求解”按钮，等待算法完成。
### 4.4 查看结果
- 扩展节点数
- 解路径长度
- 运行时间
- 最大开放集大小
- 步骤控制按钮查看解路径每一步
